# Production-Performance-Excel-Dashboard
This project presents an industrial-style Excel dashboard developed to analyze production performance and cost efficiency across multiple regions, managers, and product categories.

The dashboard uses Pivot Tables, calculated KPIs, and interactive slicers to support management-level decision-making. It enables quick identification of high-cost products, efficient managers, and regional performance trends.

#Key Features:
- Production units and cost analysis
- Cost per unit and efficiency metrics
- Region and product-based performance comparison
- Interactive Excel dashboard for business reporting

#Tools Used:
- Microsoft Excel
- Pivot Tables
- Power Query (for data cleaning)
- Data Visualization and KPI design

This project demonstrates practical data analysis and report
